This page embeds the map in an iframe.
Possible query string arguments:
css
(optional) link to an external CSS file to include
clusterRadius
max radius for marker clusters, in pixels; default=80
lat
initial latitude; default=0
lng
initial longitude; default=0
zoom
initial zoom level; default=2 (range=0-16)
gl=n
if you add this, the map will not attempt to set its initial zoom and position to the user's current location if the user is willing to allow access to it
searchZoom
how much to zoom in when the user types a location in the search bar; default=8
lc=n
if you add this, the layer-toggling control will be disabled, so the user cannot add or remove map layers
lc=h
if you add this, the layer-toggling control will be collapsed initially; by default it is expanded initially, and un-collapseable on full-width screens
amc=n
if you add this, the "add new marker" control will be disabled, so the user cannot submit new markers to be added to the map
hpe=n
If you add this, the map will not hide past events. By default, any object with a "date" field will only appear if the date is within two days of the current date and time.
layer
Which layer(s) to install on the map. If left blank, all available layers will be used. If one or more ?layer parameters are provided, then only those layers will be displayed. For example, "?layer=GROW Divestment Events&lc=n" would provide a map that only shows GROW Divestment Events (and disables the layer-toggling control since only one layer exists). "?layer=GROW Divestment Events&layer=Local Groups" would show all GROW Divestment Events and Local Groups, but no other layers. Note that the parameter(s) must match the layer names in the spreadsheet exactly, including spacing and capitalization.
actionkit
Which ActionKit campaigns to install on the map. Each campaign gets its own layer. Separate campaigns with a "|". For example, if you want to show Break Free World and People's Climate Plan Actions, it would look like this: data-map-args="actionkit=peoples-climate-plan-actions|break-free-world"
